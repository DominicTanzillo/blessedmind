name: Garden Redesign Vision

type: project
---

Garden should feel like a zen space that grows organically over time.

**Why:** The current garden has items floating randomly. User wants organized zones, tasteful arrangement, and the ability to rearrange items. Items should feel precious and earned.

**How to apply:**
- Habits at front (closest to viewer), prayers left, trophies right, pomodoros back/scattered
- Retired habits also visible, user right-clicks to remove anything
- Sparse and zen feel (40-60% empty with moss/ponds/rocks)
- Roguelike procedural generation: deterministic organic offsets from cell centers
- Desktop-only drag-drop (no mobile), positions persist to localStorage
- Suggestive decorative stone path connecting zones
- SVG art follows wabi-sabi: asymmetry, soft shapes, 2-3 hues per element, impressionistic
- Grid grows over time, never shrinks. Empty areas get zone-appropriate ground cover
